# 🚀 AssetFlow - Smart Asset Management System

AssetFlow is a web-based Asset Management System developed for the **Odoo Hackathon 2026**. It helps organizations efficiently manage employees, assets, bookings, maintenance, and asset transfers through a centralized dashboard.

---

## ✨ Features

### 👨‍💼 Employee Management
- Add Employee
- Edit Employee
- Delete Employee
- View Employee Details

### 💻 Asset Management
- Add Asset
- Edit Asset
- Delete Asset
- Assign Asset
- Asset Status Tracking

### 🔄 Asset Allocation & Transfer
- Assign assets to employees
- Transfer assets
- Expected Return Date
- Prevent duplicate assignments

### 🏢 Organization Setup
- Department Management
- Category Management

### 📅 Resource Booking
- Book shared resources
- Booking conflict detection
- Booking status tracking

### 🔧 Maintenance Management
- Raise maintenance requests
- Track maintenance status

### 📊 Reports & Analytics
- Dashboard statistics
- Total Assets
- Available Assets
- Assigned Assets
- Employee Count
- Booking Count
- Maintenance Count

### 🔔 Notifications
- Asset Assigned
- Booking Created
- Maintenance Raised
- Employee Added

### 📜 Audit Logs
- Employee Added
- Asset Added
- Asset Assigned
- Booking Created
- Maintenance Requests
- Transfer Approval

## ⚙️ Installation

Clone the repository

```bash
git clone <YOUR_GITHUB_REPOSITORY_URL>
```

Move into the project

```bash
cd AssetFlow-Odoo-Hackathon-2026
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python run.py
```

Open in browser

```
http://127.0.0.1:5000
```
